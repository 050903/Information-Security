Encryption Technologies

AES-256-GCM

Advanced Encryption Standard with Galois/Counter Mode

Process:

  1. Generate random 256-bit key
  2. Create initialization vector (IV)
  3. Encrypt data with AES-256
  4. Generate authentication tag
  5. Combine IV, ciphertext and tag
Fast Secure Industry Standard

PBKDF2-SHA256

Password-Based Key Derivation Function

Process:

  1. Generate random salt
  2. Combine password and salt
  3. Iterate hash 100,000 times
  4. Produce final key
Brute-force resistant Rainbow table proof Secure

BLAKE3

Modern Hashing Algorithm with Parallel Processing

Process:

  1. Divide data into blocks
  2. Parallel processing of blocks
  3. Combine results
  4. Produce final hash
Fast Parallel Modern

SHA-256

Secure Hash Algorithm 256-bit

Process:

  1. Divide data into blocks
  2. Process each block
  3. Combine results
  4. Produce final hash
Industry Standard Reliable Widely Used
